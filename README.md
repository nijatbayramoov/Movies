🎬 Movies Data Analysis

This project focuses on analyzing movie datasets using Python and Data Analysis techniques. The project was developed in Google Colab.

📌 Project Overview

The main purpose of this project is to explore movie data, discover patterns, and generate meaningful insights through statistical analysis and visualization.

The analysis includes:

Movie rating analysis
Popular genres comparison
Revenue and budget analysis
Vote count analysis
Trend exploration
Statistical visualizations
🛠 Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Google Colab

📈 Key Insights
Some movie genres receive higher average ratings
Popular movies tend to have larger vote counts
Revenue and budget show strong correlation
Movie popularity changes across genres and years
